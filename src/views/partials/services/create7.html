factory() and service()

  • Registers your service with the factory() or service() function
  • As others services, these functions will be called once, because services are singletons
  • DI is available for both methods
  • Functiunalities are the same, differ only on the declaration
  • Factories are handled this way by Angular:
angular.module('sample', []); .factory('UserService', ['$resource, function($resource) { return { fn1:... fn2:... }; }]); //or .. angular.module('sample', []); .factory('UserService', ['$resource, function($resource) { var UserService= function(){ this.fn1= function(){}; this.fn2= function(){}; } return new UserService(); }]); angular.module('sample', []); //Here we don't have to return anything, Angular will do the new by itself //We have won one line of code... .service('UserService', ['$resource, function($resource) { this.fn1= function(){}; this.fn2= function(){}; }]);
