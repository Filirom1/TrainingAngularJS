The service $window

  • Reference to the browser window object.
  • Can be mocked for testing, overriden or removed.
  • Always refer to this service instead of global variable window.
<div ng-app="sample" ng-controller="WindowCtrl"> <input type="text" ng-model="userName" placeholder="Enter your name..." /> <button class="btn" ng-click="showWelcomeAlert()">Login</button> </div> angular.module('sample', []) .controller('WindowCtrl', ['$scope', '$window', function($scope, $window) { $scope.showWelcomeAlert = function() { $window.alert('Hello ' + $scope.userName + "!"); } }]);
