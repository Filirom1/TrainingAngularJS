Practical Work

  1. Create a service UserService to handle the login logic and to store the logged user. Use it in your controller LoginController.
  2. Still in UserService, add a method to get the connected user if any (or undefined if not connected), and use this method in mainLayout.js (in the $scope.updateLayoutData function). If it works fine, you should see the user name in the header after the redirection.
  3. Create a utility service ProductUtils, which will contain a method called getProductRating(product). It will return the average rating for a given product.
  4. Add to ProductUtils the getRatingCss(product) method to return the correct CSS classes (used to display a product rating).
  5. Use the new service ProductUtils in the controller DetailController.
  6. Create a module named sdcoServices, and move your UserService to this new module.
  7. Make your UserService configurable by adding a debug mode parameter which can only be set during the configuration phase of the application. (when set to true, log $http success requests for example)
    To create a new module: 
    var myModule = angular.module('sdcoServices', []); // Do not forget to import this module in your app module, and to load the UserService.js BEFORE the app.js.
