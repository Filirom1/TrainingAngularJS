Security Concerns

  • Built-in protection from basic security holes, including cross-site scripting and HTML injection attacks (ie. $sanitize service).
  • XSRF protection for server-side communication.
  • Designed to be compatible with other security measures like Content Security Policy (CSP), HTTPS (SSL/TLS) and server-side authentication and authorization.
