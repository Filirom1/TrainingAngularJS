The template & templateUrl attributes

  • template: Replaces the current element with the contents of the HTML.
  • templateUrl: Same as template, but the template is loaded asynchronously from the specified URL (relative from the index page).
  • The template content is html. It has the same behavior than the HTML in your application, so you can use expressions, data-binding and other directives...
