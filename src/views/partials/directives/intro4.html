Invoking a directive

  • Directives can be placed in:
    • Element names (E).
    • Attributes (A).
    • Class names (C).
  • However, most directives are restricted to attribute only.
<span my-dir="exp"></span> <span class="my-dir: exp;"></span> <my-dir></my-dir> <!-- directive: my-dir exp --> <span ng-bind="name"></span> <!-- NOT <ng-bind="name"></ng-bind> -->
