ngShow & ngHide

  • Showhide
  • ng-show/ng-hide does not remove the elements from DOM. It uses CSS styles to hide/show elements (note: you might need to add your own classes). This way your handlers that were attached to children will not be lost.
  • ng-if creates a child scope while ng-show/ng-hide does not
<div ng-init="bShowDiv = true"> <button class="btn" ng-click="bShowDiv = !bShowDiv">Show/Hide</button> <div ng-show="bShowDiv"> I'm visible because <i>bShowDiv</i> is {{bShowDiv}}. </div> <input type="text" ng-model="login"/> <span ng-hide="!login || login.length < 7">Login is too long!!!</span> </div>
