The replace attribute

  • If false (default), the template is appended to the directive element
  • If true, the template will replace the current element
angular.module('app',[]) .directive('replaceTestFalse', function () { return { restrict: 'A', template: '<span>Directive template content</span>', }; }); angular.module('app').directive('replaceTestTrue', function () { return { restrict: 'A', replace: true, template: '<span>Directive template content</span>', }; }); <div ng-app="app"> <strong replace-test-false>Let's create a directive!</strong> <br/> <strong replace-test-true>Let's create a directive!</strong> </div>
