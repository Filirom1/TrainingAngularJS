Testing directives

Follow the steps below before testing your directives

  1. Load the template of the directive.
  2. Create an HTML element which uses your directive.
  3. Create a new scope, and provide any data needed by your directive.
  4. Use the service $compile to compile your html element with the newly created scope.
  5. You can now use the generated html to test the behavior of your directive.
describe('starRating directive unit test', function() { //Common attributes var elem, scope; // Load the directive module beforeEach(module('app')); // Load the directive template. beforeEach(module('templates/partials/myTemplate.html')); // Done before each test. Creates the tested DOM. beforeEach(inject(function($rootScope, $compile) { // Create an html snippet which uses the directive to test. elem = angular.element('<my-directive />'); //Set common attributes scope = $rootScope, scope.param = 1; // Compile the HTML snippet. elem= $compile(elem)(scope); scope.$digest(); })); it('My test', function(){ expect(elem.html().toContain('hello')); expect(scope.).toBe(2); }); });
