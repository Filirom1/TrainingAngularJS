How to use a directive

  • Directives wrap HTML fixed behavior, CSS and JS into one single element
  • The AngularJS's HTML compiler checks for directives in the DOM and then apply the specific behavior of this directive
  • To use an existing directive, you just have to add it in the DOM where you have to use it
//As an element <datepicker /> //As an attribute <input datepicker /> //And even as a class... (but not often) <input class="datepicker" />
