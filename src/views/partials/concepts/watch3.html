Use of $watch by Angular

  • Listeners using the $watch() method are registered for each view double-curly expression { { expression } }.
  • $scope properties not used in the view/template are not watched by Angular.

  • Having too many $watch can create performance issues for webpages, then use BindOnce:
    • 1.2.*: https://github.com/Pasvaz/bindonce
    • 1.3.16: natively implemented
