Controller declaration

  • Create a function with a specific name is a bad practice
  • It was allowed in previous angularjs versions, but is not anymore
  • Always use the .controller method of your Angular module
// You create your module (2 parameters means creation, // second parameters correspond to dependencies modules) angular.module('myApp', []) .controller('GreetingCtrl', ['$scope', function($scope) { //Controller content }]);
