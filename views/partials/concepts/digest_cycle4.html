The $apply function

  • Will throw following error if used in the wrong place: $apply already in progress.
  • Can be used for example in a non-Angular event handler.
<div ng-app="sample" ng-controller="ApplyCtrl"> <button class="btn" ng-click="btnClickHandler()">Click me!</button> Message: {{message}} </div> angular.module('sample',[]) .controller('ApplyCtrl', function($scope) { $scope.btnClickHandler = function() { // Will throw an error... $scope.$apply(function() { $scope.message = "Click on button"; }); }; setTimeout(function () { $scope.$apply(function () { $scope.message = "Timeout called!"; }); }, 2000); });
