Invoking a directive

  • Directives have camel cased names such as ngClick, ngBind...
  • They can be invoked by translating the camel case name into snake case with these special characters :, -, or _.
  • They also can be prefixed with x- or data- to make them HTML validator compliant.
  • Same rules for the directives parameters
<span ng-bind="name"></span> <span ng:bind="name"></span> <span ng_bind="name"></span> <span x-ng-bind="name"></span> <span data-ng-bind="name"></span> <!-- NOT <span ngBind="name"></span> -->
