Events handlers

  • ngClick: Specifies custom behavior when an element is clicked.
  • ngChange: Evaluate given expression when user changes the input
  • ngDblclick: Custom behavior on double click event.
  • ngMouseDown, ngMouseEnter, ngMouseLeave ngMouseOver, ngMouseUp: Mouse events.
<button class="btn" ng-click="txtEvent='click'">Click event</button> <input type="text" ng-model="login" ng-change="txtEvent='change'" placeholder="Edit input to trigger change event..." /> <button class="btn" ng-mouseover="txtEvent='mouseover'">Mouse events</button> Event: {{txtEvent}}
