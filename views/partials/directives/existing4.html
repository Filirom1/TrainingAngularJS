ngIf

  • Removes conditionally a portion of the DOM tree
  • If ngIf removes an element from the DOM, remember that handlers associated to this elements are removed as well, which means that you will have to reattach them if needed
<div ng-init="bRemoveDiv = true"> <button class="btn" ng-click="bRemoveDiv = !bRemoveDiv"> Show/Remove </button> <div ng-if="bRemoveDiv"> I'm visible because <i>bRemoveDiv</i> is {{bRemoveDiv}}. </div> </div>
