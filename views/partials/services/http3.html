Main Angular services: $http

Setting HTTP Headers

For all requests:

  • $httpProvider.defaults.headers.common (common for all requests).
  • $httpProvider.defaults.headers.post (POST requests).
  • $httpProvider.defaults.headers.put (PUT requests).

For one request:

$http.get('/users', {headers: {'Authorization': 'Basic QWxhZGRpbjpvcGVuIHNlc2FtZQ=='}});
