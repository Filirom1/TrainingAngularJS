Practical Work

  1. In our ecommerce application, create a service UserService to handle the login logic, and to store the logged user. Use it in your controller LoginController.
  2. Create a utility service ProductUtils, which will contain a method called getProductRating(product). It will return the average rating for a given product.
  3. Then make another method getRatingCss(product), which will use the previous one to return the correct CSS class used to display a product rating.
  4. Use the new service ProductUtils in the controller DetailController.
  5. Create a module called sdcoServices, which will contain your services UserService and User (see User.js).
  6. Add to your service UserService a debug mode, which will determine if logs are displayed in $http handler functions (using the $log service). Make sure this parameter can be ONLY set during the application startup.
    To create a new module: 
    var myModule = angular.module('sdcoServices', []); // Do not forget to import this module in your app module, and to load the UserService.js BEFORE the app.js.
