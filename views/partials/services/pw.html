Practical work

  • In the home page of our ecommerce application, we will display all the news added by the webmasters. Use the $resource service to manage news. Following functionalities are needed:
    1. look at jetty-resteasy/src/main/java/com/example/rest/services/NewsApi.java to understand the rest api (don't hesitate to ask the former if you need). All REST urls start by /api (for example, /api/news is an available operation...)
    2. List all the news in the table element.
    3. Displays the news of the day in the bottom panel.
      • First, the selected news will be the one whose id is equals to 1.
      • Second, the selected news will be a random one (see server.js)
    4. You can "like" a news, which will increase the value of its likes property.
    5. Delete a news.
    6. Add a news.
  • Tips:
    1. Understand that this url is enough to do all needed operations: /api/news/:operation/:id
