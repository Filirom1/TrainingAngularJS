Main Angular services: $http

  • Facilitates communication with remote HTTP servers via XMLHttprequest or JSONP.
  • Based on the deferred/promise API.
  • You can enable cache to store the response in local cache.
<div ng-controller="HttpCtrl"> File content: {{result}} </div> // exoController is a custom method to get dynamic behavior // In a standard app, you would use controller instead... angular.module('trainingAJ') .exoController('HttpCtrl', ['$scope', '$http', function($scope, $http) { $scope.result = "Loading file..."; $http({method: 'GET', url: 'data/exos/users.json'}). success(function(data, status, headers, config) { $scope.result = data; }). error(function(data, status, headers, config) { $scope.result = "Error loading file."; }); }]);
