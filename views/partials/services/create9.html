Summary

  • value or constant for an existing value.
    • value for an independent object/function. Can be decorated.
    • constant for configuration data. Can't be decorated.
  • factory, if you need another service. This is the most common way to create services.
  • service, if you love Java and want to use object notation, passing a constructor which will be constructed using the new keyword.
  • provider, if you need a configurable service.
