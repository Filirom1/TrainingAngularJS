Main Angular Filters

  • currency: Formats a number as a currency (ex: $1,234.56). If no currency symbol is specified, default symbol for current local is used.
  • date: Formats date to a string with specified format.
  • filter: Filters items from an array and returns them as a new array.
  • json: Converts a JS object into JSON String.
  • limitTo: Returns an array with a limited number of elements.
  • lowercase/uppercase: Converts string to lowercase/uppercase.
  • number: Formats a number as text (or an empty string if not a number).
  • orderBy: Orders an array by the specified expression. Optional parameter reverse can be used to reverse the order of the array.
