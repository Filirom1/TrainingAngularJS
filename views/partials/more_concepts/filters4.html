Creating Angular Filters

  • You can register new filters in your application, using filter() module function.
  • It returns a filter function, which takes the value to filter as the first argument. Then, all filter parameters are passed as additional arguments.
<div ng-app="sample"> {{ "Deinstitutionalization" | split:10 }} <br/> {{ "Deinstitutionalization" | split:5:true }} <br/> </div> angular.module('sample', []) .filter('split', function() { return function(input, nbChars, uppercase) { var out = input; if (input.length > nbChars) { out = input.substring(0, nbChars) + "..."; } if (uppercase) { out = out.toUpperCase(); } return out; }; });
