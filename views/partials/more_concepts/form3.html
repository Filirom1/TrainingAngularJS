Form Validation - methods and css

  • $scope.myForm.$setDirty(): Changes form state to $pristine
  • $scope.myForm.$setPristine(): Changes form state to $pristine
  • $scope.myForm.element.$setValidity(ruleName, isNotValid): Defines if the rule ruleName (required, min, etc.) has been validated (false) or not.
  • Use predefined CSS classes on your form elements:
    • ng-valid/ng-invalid if the model is valid or not
    • ng-valid-[key]/ng-invalid-[key] if the property defined with $setValidity is valid or not
    • ...
  • More details:
    • https://docs.angularjs.org/api/ng/type/form.FormController
    • https://docs.angularjs.org/guide/forms
